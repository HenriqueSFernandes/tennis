# Rio Tinto Tennis

> **Disclaimer:** This project was fully vibe coded, I did not look at a single line of code. It was created out of necessity and as a test of what current AI tools are capable of.

## What is this?

Rio Tinto has 2 free public tennis courts, but booking them through the official website is a painful experience, the UI is buggy, slow, and frustrating to use. On top of that, each booking slot requires a separate account, meaning that booking 2 consecutive hours means juggling 2 different logins.

This is a self-hosted web app that wraps the official booking system into a clean, usable interface and centralizes all your accounts in one place.

## Features

- Manage multiple riotinto.pt accounts from a single dashboard
- View the weekly schedule for both courts and book available slots
- See your current reservations and cancel them

## Future Plans

- Move from a single master password shared across all accounts to a proper per-user system where each user manages their own accounts
- Create new riotinto.pt accounts directly through this app, without having to visit the official website
